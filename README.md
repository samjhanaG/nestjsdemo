Installation Guide

Follow these steps to set up and run the NestJS project locally.

1. Clone the Repository

git clone <https://github.com/samjhanaG/nestjsdemo>

2. Navigate to the Project Folder

cd app

3. Install Dependencies

npm install

4. Install NestJS CLI (Optional)

If NestJS CLI is not installed globally:

npm install -g @nestjs/cli

5. Run the Development Server

npm run start:dev

The server will start on:

http://localhost:3000

6. Build the Project

npm run build

7. Run Production Build

npm run start:prod

Requirements
Node.js
npm
NestJS CLI
Git
Verify Installation

node -v
npm -v
nest -v
